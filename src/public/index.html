A Simple NestJS Starter Kit

A REST API starter kit based on the NestJS framework and written in Typescript, featuring JWT-based authentication, PostgreSQL integration using TypeORM, and Docker for containerized development and deployment.

Description

This comprehensive starter kit can be used as a foundation for creating your own REST APIs.
Key features include:

  • Built using the NestJS framework and Express under the hood
  • Written in TypeScript for type safety and developer productivity
  • JWT-based authentication system with secure token management
  • Two-Factor Authentication (2FA) support via email
  • PostgreSQL integration via TypeORM
  • Dockerized development environment for consistency across machines
  • Environment variable management and run-time validation for reliability across environments
  • Token rotation with hashed storage for enhanced security
  • Modular architecture following NestJS best practices for maintainability and scalability
